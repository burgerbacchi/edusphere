# EduSphere

EduSphere is an interactive and user-friendly educational platform designed to streamline and enhance the learning experience. It features a dynamic login page with school-themed elements, intuitive navigation, and an option for new user registration. This platform aims to provide seamless access to educational resources and tools.

---

## Features

### Login Page
- **User Authentication**: Secure login for registered users.
- **Navigation Links**:
  - **Dashboard**: Access user-specific resources and activities.
  - **Contact Us**: Reach out to the EduSphere support team.
  - **About Us**: Learn more about the EduSphere platform.
- **Registration**: New users can create an account easily.
- **School Theme**: Visually appealing design with educational elements.

## Installation and Setup

### Prerequisites
Ensure you have the following installed:
- Node.js (v14 or higher)
- npm or yarn

### Steps
1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/edusphere.git
   ```
2. Navigate to the project directory:
   ```bash
   cd edusphere
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm start
   ```
5. Open your browser and visit:
   ```
   http://localhost:3000
   ```
